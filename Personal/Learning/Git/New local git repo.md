Add to GitHub[^1]
***
#gitnewrepo #newrepo #repo

1. Go to the current root data source folder
```bash
# main for main branch
git init -b main
```

2. Add the files to stage
```bash
git add .
```

3. Commit them
```bash
git commit -m "Initial Commit"
```
