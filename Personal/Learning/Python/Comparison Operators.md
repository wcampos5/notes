#### Compare two values

|Operator|Description |
|--------|-----------|
|\=\=| Equal
|!=| Different|
|>| Major than|
|>=| Major or equals to|
|<| Minor than|
|<=| Minor or equals to|


> [!code]
> `saldo = 450`
> `saque = 200`

`print(saldo == saque)`
>[!failure] False

`print(saldo != saque)`
> [!success] True

`print(saldo > saque)`
> [!success] True

`print(saldo >= saque)`
> [!success] True

`print(saldo < saque)`
>[!failure] False

`print(saldo <= saque)`
>[!failure] False
