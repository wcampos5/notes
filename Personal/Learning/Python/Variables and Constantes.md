
#### Variables
The type will be automatically assigned based on attributed value

> [!example] Int
> age = 28

> [!example] String
> name = 'Wilson'

#### Constants
Python doesn't have constant

By convention **constants** are declared in **UPPER CASE**
> [!example] Constant examples
> ABS_PATH = '/home/wcampos5/'
> DEBUG = True
> STATES = [
> 	\'SP\',
> 	\'RJ\',
> 	\'BA\',
> 	]
> AMOUNT_TAX = 0.1
